Community chat
the server can't read.

Discreet is an open-source platform for communities, teams, and friends. Servers, channels, voice, AI agents, file sharing. One difference: every message is encrypted before it leaves your device. The server stores ciphertext it mathematically cannot decrypt.

How it works

Zero knowledge by design.

The server relays encrypted blobs between authenticated clients. It never possesses decryption keys. This isn't a policy, it's an architectural constraint.
