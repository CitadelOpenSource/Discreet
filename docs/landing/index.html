🔐

Zero-Knowledge Encryption

AES-256-GCM with MLS (RFC 9420) group keys. Messages encrypted in your browser before they leave your device. The server stores ciphertext it literally cannot decrypt.

🎙️

Voice, Video, Screen Share

WebRTC peer-to-peer voice channels with OBS-grade audio processing: noise gate, compressor, expander, 5-band EQ, per-user volume. Screen sharing, camera, soundboard.

💬

Text, Forums, Threads

Text channels, forum channels with threaded discussions, DMs, group DMs. GIF picker, custom emoji, @mentions, reactions, pins, inline media, URL previews.

📺

Watch Together

Sync YouTube playback across your entire voice channel. Soundboard with custom clips. Voice messages. Activities and events with RSVP.

🤖

AI Agents (Patent-Pending)

Specialist AI agents auto-spawn as encrypted group members. Legal, medical, security experts — all conversations E2EE. The server never sees what you asked.

🏠

Self-Hostable

One-command deployment. Runs on a Raspberry Pi ($50 hardware, $5/year electricity) or Oracle Cloud free tier. Docker Compose + automatic HTTPS.

⚙️

60+ Settings

Appearance, audio, video, accessibility, keybinds, privacy, notifications — VLC/OBS level customizability. Dyslexia font, reduce motion, zoom, rebindable shortcuts.

🛡️

Moderation Without Surveillance

Client-side auto-mod rules. Zero-knowledge word search. Audit logging with SHA-256 hash-chain integrity. No message scanning. No AI training on your data.

📖

Open Source (AGPL-3.0)

Every line of code is auditable. Rust backend. React client. 14,000+ lines of code. No proprietary black boxes. No trust-us-bro security. Verify everything.
