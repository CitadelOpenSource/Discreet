What we're building

Everything Discord does. Nothing Discord knows.

🔐

End-to-End Encrypted Text

Every message encrypted with MLS (RFC 9420) before it leaves your device. The server stores ciphertext it cannot decrypt. Not a policy — a mathematical guarantee.

Signal-grade encryption
🎙️

Encrypted Voice Channels

Low-latency voice chat with end-to-end encryption via WebRTC Insertable Streams. The server relays encrypted audio it cannot hear.

E2EE voice & video
🏰

Servers, Channels, Roles

The community features you know: text channels, voice channels, categories, roles, permissions, invites. Familiar UX, fundamentally different architecture.

📱

Multi-Device Sync

Use Discreet on desktop, phone, and web simultaneously. Each device has its own encryption keys. Messages arrive everywhere, encrypted for each device individually.

🔍

Private Search

Full-text search across all your messages — running entirely on your device. Search queries never touch the server. Your conversations stay yours.

🌐

Self-Hostable

Run your own Discreet instance with one command. Your server, your rules, your data. Or use our hosted service — either way, we can't read your messages.

docker compose up
Zero-knowledge architecture

The server is a relay, not an oracle.

Discreet is built on the principle that the server should be treated as hostile. All cryptographic operations happen on your device.

Client
Tauri Desktop (Win/Mac/Linux) React Native (iOS/Android) Web Client (any browser) All encryption happens here
Protocol
MLS (RFC 9420) — group E2EE Signal Protocol — DM E2EE WebRTC SRTP — voice E2EE TLS 1.3 — transport
Server
Rust (Axum) PostgreSQL Redis pub/sub Sees only encrypted blobs
Infra
Docker self-hosted Tor-compatible AGPL-3.0 open source CoTURN voice relay
